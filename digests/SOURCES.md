# DAO Digest — Sources Registry

Purpose: keep a growing list of places where DAOs *actually* coordinate (forums, Snapshot/Tally, docs, repos, blogs, calls). This is used by the daily DAO Digest.

**Coverage goal:** as many worthwhile DAOs and ecosystems as possible (Ethereum L2s + non-EVM like Solana/Cosmos/etc.). Output stays English even if sources are not.

## How we’ll use this
Governance is the filter. Anything can be in-scope if it affects:
- proposal flow (temperature checks → votes → execution)
- treasury/risk/security decisions
- delegation systems and voting power
- governance tooling/ops

- **Tier 1 (always check):** high-signal venues with frequent governance activity.
- **Tier 2 (rotate):** check a subset daily to go wider.
- **Tier 3 (watchlist):** niche / emerging DAOs; check weekly unless something pops.

**Language policy:** sources can be any language; the digest is always written in English.

Add new sources as we discover them.
