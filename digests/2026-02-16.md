Top governance moves (verified / high-confidence)

1) Optimism: “Align OP Token with Superchain Success” — SUCCEEDED
Optimism’s Token House / Citizens’ House joint process just approved a 12‑month program that dedicates 50% of incoming Superchain revenue to buying back OP, starting in February. The OP is intended to be held in the Collective treasury, alongside remaining sequencer ETH, with monthly ETH→OP conversions via an OTC provider.

Primary links:
- Vote result (Agora): https://vote.optimism.io/proposals (open “Proposal to Align OP Token with Superchain Success”)  
- Proposal detail + context: https://gov.optimism.io/t/proposal-to-align-the-op-token-with-superchain-success/10527

Why it matters: This is one of the cleaner “token value ↔ network cashflow” governance experiments we’ve seen on a major L2. It’s also a live test of whether DAOs can execute buybacks without turning governance into quarterly earnings calls.

2) TheDAO Security Fund: activating idle recovery-era ETH for Ethereum security
A decade after TheDAO drama, a new “TheDAO Security Fund” effort is positioning the unclaimed / edge-case recovery funds as an Ethereum security endowment. The headline: 75,000+ ETH (over $220M) to fund audits, incident response, wallet UX security, protocol security work, and more—using open funding mechanisms (quadratic/retro/RFPs/etc.).

Primary links:
- Project site: https://thedao.fund/
- Long-form write-up (Paragraph): https://paragraph.com/@thedao.fund/thedao-security-fund-activating-75000-eth-for-ethereum-security

Notable detail (from the write-up): they emphasize this does NOT touch the “main DAO withdraw contract” enshrined by the fork; it focuses on the edge-case contracts and intends to keep claims open.

3) ENS: “Enable Root and Registrar Security Controllers” (still contentious)
The proposal to expand security controller powers—effectively giving a security council more “break-glass” authority over root/registrar operations—is still a live lightning rod for the security-vs-decentralization trade.

Primary link:
- ENS forum thread: https://discuss.ens.domains/t/executable-enable-root-and-registrar-security-controllers/21872

Why it matters: This pattern keeps repeating across DAOs: the moment a protocol matures into real infrastructure, governance wants an emergency steering wheel. The debate is always “who holds it, how transparent is it, and what’s the rollback story when the emergency ends?”

Down in the weeds (3–6 quick hits)
- Optimism’s buyback design choice to use an OTC provider for monthly ETH→OP conversions is a subtle “we’re optimizing market impact” decision. It’s worth watching who the provider is, what the reporting looks like, and what the governance audit trail is for each conversion.
- TheDAO Security Fund write-up explicitly calls out operational expectations: large onchain moves, signer updates, and staking flows. That’s catnip for chain sleuths—expect viral “whale alert” moments that are operational, not adversarial.
- TheDAO plan leans hard on “funding rounds run by independent operators.” That’s good governance hygiene, but it also creates a second-order governance problem: operator selection becomes the real battleground.
- The ENS proposal (if it moves forward) is another example of “council power creep” that starts defensively and ends as precedent. The only durable safety valve is strict scope + transparent triggers + sunset clauses.

Tooling / infra (2–4)
- Agora continues to be the default governance UX layer for Optimism (and an increasing number of ecosystems). The “Connect wallet to vote” gate is normal, but the important bit is the public record: outcomes, quorum, thresholds.
- Expect more DAOs to operationalize “security work” funding as a first-class budget line (not a one-off audit grant). TheDAO Security Fund is an extreme example (endowment-style), but the pattern is spreading.
- Realms (Solana governance) is pushing clearer governance mental models in public. Not tooling in the strict sense, but it does shape how communities accept (or reject) new layers of authority.

Themes (2–4)
- Governance is rediscovering cashflows: buybacks, endowments, and “how do we fund security forever?”
- “Break-glass” authority is the recurring debate: councils and controllers keep reappearing as protocols become critical infrastructure.
- Quorum economics is a security topic now, not just a turnout metric.
