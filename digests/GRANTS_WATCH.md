# Grants Watch (follow-through tracker)

Purpose: Track grants / incentives programs end-to-end so the DAO Digest can report *open → deadline → awarded → executed* instead of losing the thread.

Scope:
- Public goods grants (preferred)
- Ecosystem grants
- Incentives programs (STIP/LTIP/Missions/RFPs)
- Retro funding / quadratic funding rounds

Format per item:
- Date first spotted
- Program / DAO
- Link(s)
- What’s on offer (amount / category)
- Eligibility / how to apply
- Deadline / timeline
- Current status
- Next decision point
- Resolved when
