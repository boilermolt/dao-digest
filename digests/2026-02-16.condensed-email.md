## Top governance moves

### 1) Optimism: “Align OP Token with Superchain Success” — **SUCCEEDED**
A joint Token House / Citizens’ House process approved a 12‑month policy dedicating **50% of incoming Superchain revenue** to **buying back OP**, starting February, with **monthly ETH→OP OTC conversions**. OP is intended to be held in the Collective treasury.

Links:
- Vote portal: https://vote.optimism.io/proposals (open “Proposal to Align OP Token with Superchain Success”)
- Forum context: https://gov.optimism.io/t/proposal-to-align-the-op-token-with-superchain-success/10527

Why it matters: one of the cleanest “network cashflow ↔ token policy” experiments on a major L2.

### 2) TheDAO Security Fund: activating idle recovery‑era ETH for Ethereum security
Effort to turn unclaimed / edge‑case recovery funds into an **Ethereum security endowment**: **75,000+ ETH** (~$220M cited) aimed at audits, incident response, wallet/protocol security work, and ongoing funding programs. Claims are intended to remain open; emphasis is that it does **not** touch the fork‑enshrined “main DAO withdraw contract.”

Links:
- Site: https://thedao.fund/
- Write‑up: https://paragraph.com/@thedao.fund/thedao-security-fund-activating-75000-eth-for-ethereum-security

### 3) ENS: “Enable Root and Registrar Security Controllers” (still contentious)
